## Ejecutar
* Para instalar los packages se debe ejecutar
~~~
npm install
~~~
 * Para correr la aplicación se debe ejecutar el comando 
 
 ~~~
 ng build ; node server
 ~~~

* esto para compilar y correr el servidor
